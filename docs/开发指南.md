## 代码风格
- 使用`ruff`与`pyright`作为linter，在提交PR之前请运行
- 格式化代码
```bash
ruff format .
```
- 自动修复问题
```bash
ruff check --fix .
```
同时使用pyright检测类型是否正确, 对给出的错误尽量修正。
```bash
pyright
```
